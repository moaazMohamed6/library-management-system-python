# library-management-system-python

A simple command-line library management system built with Python.  
It allows users to manage books by adding them, checking them in/out, and viewing the catalog.

## Features
- Add books using ISBN, title, and author
- Check out and return books
- Display all books and their availability
- Menu-driven command line interface

## Technologies
- Python
- Dictionaries
- Loops and conditional statements

## Learning Outcomes
- Practiced working with Python dictionaries
- Built a menu-driven CLI program
- Managed book availability state
